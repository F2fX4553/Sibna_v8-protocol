# Web Client

React-based web client for Sibna secure messaging.

## Development

```bash
# Install dependencies
npm install

# Build WASM first
cd ../wasm && npm run build && cd ..

# Start development server
npm run dev
```

## Features

- End-to-end encrypted messaging
- Group messaging support
- Identity key management
- Real-time chat interface

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- WebAssembly
