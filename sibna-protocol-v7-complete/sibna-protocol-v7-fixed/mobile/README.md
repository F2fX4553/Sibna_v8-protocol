# Mobile SDKs

This directory contains mobile SDKs for Sibna Protocol.

## Platforms

### Android (Kotlin)
- Path: `android/`
- Language: Kotlin
- Min SDK: Android API 24+

### iOS (Swift)
- Path: `ios/`
- Language: Swift
- Min Version: iOS 14.0+

## Building

### Android
```bash
cd android
./gradlew build
```

### iOS
```bash
cd ios
swift build
```

## Features

- Full protocol support (X3DH, Double Ratchet)
- Group messaging
- Identity management
- Secure key storage
- Cross-platform API consistency
